1-6 希尔排序

介绍​

希尔排序也是插入排序的一种。他是对直接插入排序的一种版本的改进。它的核心思维是先将整个待排元素序列分割成若干个子序列(一般元素的个数/2获得)，然后对每个组分别进行直接插入排序。随着序列的逐渐减小，所包含的记录就越来越多，当序列为1的时候，整个数据合成一组。 则完成排序。

说明​

希尔排序因为其排序的特殊性，对中等大小规模的数据排序表现良好。但是对非常大型的数据排序不是最优的选择。

排序示例​

希尔排序目的为了加快速度改进了插入排序，交换不相邻的元素对数组的局部进行排序，并最终用插入排序将局部有序的数组排序。

在此我们选择以 gap=length/2的方式类进行排序。用序列 {n/2,(n/2)/2…1} 来表示。

初始数据​

int  a[] = {9,1,2,5,7,4,3,6};
int len = sizeof(a)/sizeof(a[0]); // 8

第一次排序， gap = 8 / 2 = 4​

第二次排序， gap = 4 / 2 = 2​

第三次排序， gap = 2 / 2 = 1​

  • shell_sort.c
//
// Created by gaosx on 22-9-9.
//

#include <stdio.h>

void shell_sort(int *p, int n) {
    int i = 0, j = 0, k = 0, temp = 0;
    for (k = n / 2; k >= 1; k = k / 2) //分成的组数
    {
        for (i = k; i < n; i++)//插入排序的次数
        {
            temp = p[i];
            for (j = i - k; j >= 0 && p[j] > temp; j = j - k)//找到每组中的元素，进行排列。
            {                                             //每组元素相隔k个
                p[j + k] = p[j];
            }
            p[j + k] = temp;
        }
    }
}

void output_array(int *p, int n) {
    int i = 0;
    for (i = 0; i < n; i++) {
        printf("%d ", p[i]);
    }
    printf("\n");
}

int main() {
    int a[] = {9, 1, 2, 5, 7, 4, 3, 6};

    int n = sizeof(a) / sizeof(a[0]);

    shell_sort(a, n);
    output_array(a, n);
    return 0;
}
