概述

嵌入式开发定义​

  • 通俗的讲，嵌入式指两个方面，一个是为某一个特定行业特定场景定制的硬件设备
  • 另一个是为这个硬件设备开发的专用软件系统。这个软件系统我们称之为嵌入式系统

为什么要选择嵌入式行业​

  • 一句话：蓝海新风口、高薪稳定不内卷

嵌入式行业-国家策略​

  • 国产化趋势
    • 医疗行业90%以上的医疗检测设备都是由国外提供
  • 改变缺"芯"少"魂"的局面
    • 中国大陆芯片全球占比仅为4%
  • 国产操作系统
    • 鸿蒙、欧拉、UOS、麒麟等，大部分采用开源操作系统内核Linux
  • 新能源汽车

软件开发内卷/嵌入式人才供不应求​

web软件开发（java、前端）​

  • 语言入门简单，精通难（背后原理）很多框架做了大部分工作，开发人员更多的是做增删改查
  • 平台构建，让小公司不需要再单独开发管理软件
  • 人才愈加饱和，出现了大量裁员潮

物联网/智能硬件/嵌入式开发​

  • C/C++语言
  • 以硬件产品为导向，软件为硬件产品定制，不能通用
  • 有一定门槛，出现了需求量大，人才太少的局面
