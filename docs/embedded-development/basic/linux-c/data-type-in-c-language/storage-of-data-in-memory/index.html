数据在内存的存储

数据类型分类​

数据类型细分​

有符号和无符号​

我们上述，所介绍的数据类型，全部都可以分为两种，分别用两个关键字对应。

signedunsigned
含义有符号数据 【正数,0,负数】无符号数据 【正数，0】
常用写法signed intunsigned int
特点可省略不可省略

示例

signed char  a ;   //char  a;
unsinged char b;

signed int c;     //int c;
unsinged int  d;

8bit数据的存储​

char，有符号类型，占1bytes。可以表示负数，0，正数。范围[-128~127] -128~-1 0~127

unsigned char，无符号类型，占1bytes。可以表示0，正数。范围[0~255]

1bytes = 8bit     ===>  0000 0000    0 
                    ~
                        1111 1111   255

超出范围数据的计算方法​

数据范围​

char [-128~127] 正数，负数，0

unsigned char [0~255] 正数，0

设计规则​

无符号类型的数原码，反码，补码是它本身。 [正数和0]

有符号类型的数的最高位表示符号位，0表示为正数，1表示为负数

负数的反码 = 符号位不变，其他位按位取反

负数的补码 = 反码 + 1

计算方法​

先计算整数的补码(=右边的数)，把补码赋值给变量，然后观察变量的数据类型。若是为unsinged char类型，一定为正数或0，原，反，补一样。%d输出的原码就是补码，直接转换为元素输出即可。若是为char类型，观察变量内存存储数据的最高位，1表示为负数，%d输出需要转换为原码输出。

示例用法​

示例用法1​
#include <stdio.h>

int main()
{
    char  a = 200; 
    printf("a = %d\n",a);  //-56
    return 0;
}         
         
//计算方法：
//200 = 128 + 64 + 8  ===> 1100 1000 原码
//            200的补码===> 1100 1000 ,复制给了变量a.
//那么a在内存中的补码: 1100 1000 ,而a为char类型，最高位表示符号位，为1，说明负数
                      // 取反: 1011 0111 
                      //    +1  : 1011 1000 ===>-(0011 1000)==>-0x38====>-56
示例用法2​
#include <stdio.h>
int main()
{
    unsigned char b = 280;
    printf("b = %d\n",b); //24
    return 0;
}       

//计算方法:
//280 = 256 + 16 + 8 ==> 1 0001 1000,b为unsigned char类型，占8位，
          //故只能把低8bit复制给变量b。
          // b在内存中的补码: 0001 1000   ,b为unsigned char类型，一定为0，或者正数
          //           原码: 0001 1000  ====>0x18 ===>16 + 8 = 24

练习​

char a = 180;

unsigned char b = 300;

要求以上数据用%d输出，请大家用笔算了之后，再通过代码验证。

代码​

#include <stdio.h>
int main()
{
    char a = 180;
    unsigned char b = 300;
    printf("a = %d\n", a); // -76
    printf("b = %d\n", b); // 44
    return 0;
}

// 计算方法：
// 180 = 128 + 32 + 16 + 4
// 180的原码：1011 0100
// 180的补码：1011 0100
// 最高位是1，说明负数
//      反码：1100 1011
//      补码：1100 1100
//      ====》 -(0100 1100) => -76
//
// 300 = 256 + 32 + 8 + 4
// 300的原码：0010 1100
// 300的补码：0010 1100
//      正数
//      反码：0010 1100
//      补码：0010 1100
//      ===》 2^5 + 2^3 + 2^2 = 44
//

输出​

a = -76
b = 44
